Frame handle_gps_power_status ( const std::string & param,
OperationType operationType )

Handler for controlling GPS module power state.

Parameters
paramFor SET: "0" to power off, "1" to power on. For GET: empty
operationTypeGET to read current state, SET to change state
Returns
Frame containing:
  • Success: Current power state (0/1) or
  • Error: Error reason
Note
KBST;0;GET;7;1;;TSBK
Return current GPS module power state: ON/OFF
KBST;0;SET;7;1;POWER;TSBK
POWER - 0 - OFF, 1 - ON
